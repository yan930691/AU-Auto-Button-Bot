## ✨ Features

- 📢 Auto post text, photo, video & documents  
- 🔘 Inline button attachment system  
- 👮 Owner + Admin control system  
- 📊 Live stats (users & posts tracking)  
- 📢 Broadcast system for users  
- 🧾 Admin management (add / remove / list)  
- 📁 Log channel integration  
- ⚡ Fast and lightweight Pyrogram bot  
- 🔐 Secure permission system  
- 🌐 Render & Koyeb deployment supported

## 📌 Commands

```txt
start        - Start Bot
stats        - Show bot statistics (Owner & Admin)
broadcast    - Send message to users (Owner & Admin)
addadmin     - Add admin (Owner only)
removeadmin  - Remove admin (Owner only)
admins       - List admins (Owner only)
```
